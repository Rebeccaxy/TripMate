# Popular Places 图片资源文件夹

此文件夹用于存放Popular Places板块的地点图片资源。

## 使用说明

### 地点图片
- `PP1.png` - Mount Bromo 图片
- `PP1@2x.png` - 2倍图（可选）
- `PP1@3x.png` - 3倍图（可选）

- `PP2.png` - Labengki Sombori 图片
- `PP2@2x.png` - 2倍图（可选）
- `PP2@3x.png` - 3倍图（可选）

- `PP3.png` - 第三张地点图片
- `PP3@2x.png` - 2倍图（可选）
- `PP3@3x.png` - 3倍图（可选）

## 使用方式

在代码中通过以下方式引用：
```typescript
require('@/assets/images/popular-places/PP1.png')
require('@/assets/images/popular-places/PP2.png')
require('@/assets/images/popular-places/PP3.png')
```

## 注意事项

- 建议使用PNG格式以保证图片质量
- 图片尺寸建议适配卡片尺寸（160x200px）
- 如果使用@2x和@3x版本，系统会自动根据设备分辨率选择合适的图片
