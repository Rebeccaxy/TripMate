# Home界面图片资源

此文件夹用于存放Home界面所需的图片资源。

## 需要的图片文件

### 背景图
- `home-bg.png` - Home界面的背景图（覆盖整个界面）
- `home-bg@2x.png` - 2倍图（可选，用于高分辨率屏幕）
- `home-bg@3x.png` - 3倍图（可选，用于超高分辨率屏幕）

## 使用方式

在代码中通过以下方式引用：
```typescript
require('@/assets/images/home/home-bg.png')
```

## 注意事项

- 建议使用PNG格式以保证图片质量
- 背景图尺寸建议适配不同屏幕尺寸
- 如果使用@2x和@3x版本，系统会自动根据设备分辨率选择合适的图片
